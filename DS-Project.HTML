Project Title: OTT Platform




Description

The OTT Platform Movie Management System is a Java-based console application designed to manage and display information about movies available on an OTT platform. The system allows users to view available movies, check movie details such as genre, ratings, and number of views, and organize movie information efficiently. This project demonstrates the use of object-oriented programming concepts to build a simple movie management system.
Technologies Used

- Java
- Object-Oriented Programming (OOP) concepts
- Arrays and Classes
Features

- Stores multiple movie details in the system
- Displays available movies with their information
- Shows movie name, genre code, ratings, and views
- Uses classes and objects to manage movie data
- Simple menu-driven console interface

What I Learned

Through this project, I learned how to design Java classes to represent real-world entities like movies, store multiple objects using arrays, and organize movie-related data using object-oriented programming concepts.
