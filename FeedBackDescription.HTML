Project Title: FeedBack Form

Description

The College Feedback System is a web-based form designed to collect feedback from students about their academic experience. The system allows users to submit their name, email, department, rating, and comments through a simple and user-friendly interface. This project demonstrates how web forms can be used to gather and organize feedback efficiently.
Technologies Used

- HTML
- CSS
Features

- User-friendly feedback form
- Input fields for name and email
- Dropdown selection for department and rating
- Text area for detailed comments
- Submit and Reset buttons for form control

What I Learned

Through this project, I learned how to create structured web forms using HTML, style them using CSS, and design a simple user interface for collecting feedback data.
