Project Title: My Portfolio

Description

My Portfolio is a personal website created to showcase my projects, skills, and personal information. The website is designed with a clean and simple layout to make navigation easy for users. It helps present my work and achievements in an organized way.
Technologies Used

- HTML
- CSS
- Basic JavaScript
Features

- Responsive and clean user interface
- Sections like Home, Skills, Projects, and Contact
- Simple and easy navigation
- Displays personal projects and skills

What I Learned

Through this project, I improved my understanding of HTML structure, CSS styling while building a simple and user-friendly website.
