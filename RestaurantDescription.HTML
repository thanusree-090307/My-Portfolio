Project Title: Restaurant Menu

Description

The Bhojan Nivas Restaurant Menu is a simple webpage designed to display food items and their prices in a clear and organized layout. The project is built using HTML and CSS, featuring images of dishes, item names, and prices arranged in a structured format. The page demonstrates basic front-end development skills and focuses on creating a clean and user-friendly menu interface.
Technologies Used

- HTML
- CSS
Features

- Displays restaurant menu items with images
- Shows food name and price clearly
- Simple and clean layout design
- Organized menu sections for better readability

What I Learned

Through this project, I learned how to structure web pages using HTML, style elements using CSS, and organize content visually to create a simple restaurant menu interface.
