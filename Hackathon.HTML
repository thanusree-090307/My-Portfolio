Project Title: Disaster Relief Resource Managment




Description

The Disaster Relief Resource Management System is a Java-based console application designed to manage and distribute essential resources during disaster situations. The system allows users to view affected areas, distribute relief resources such as food, water, medical kits, and shelters, track available resources, and generate reports. The application helps simulate how disaster management teams allocate resources efficiently to affected locations.
Technologies Used

- Java
- Object-Oriented Programming (OOP) concepts
- File Handling
Features

- Menu-driven console application
- Displays multiple affected areas such as Hyderabad, Chennai, Bangalore, Mumbai, and Delhi
- Allows selection of different disaster types (Flood, Earthquake, Cyclone, Fire)
- Tracks available resources in each affected area
- Generates a disaster relief report and saves it to a text file

What I Learned

Through this project, I learned how to apply Object-Oriented Programming principles in Java, manage multiple classes in a project and design a menu-driven console application for resource management.
