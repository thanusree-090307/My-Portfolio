Project Title: NearNest Website




Description

NearNest is a simple web application designed to connect customers, shopkeepers, and delivery partners on a single platform. The website allows customers to browse nearby products, shopkeepers to manage their items, and delivery partners to handle order deliveries. The interface is designed to be clean and user-friendly, making navigation easy for different types of users.
Technologies Used

- HTML
- CSS
- Basic JavaScript
Features

- Separate interfaces for Customer, Shopkeeper, and Delivery Partner
- User-friendly login and navigation system
- Product browsing and order placement for customers
- Shop management features for shopkeepers
- Delivery tracking functionality for delivery partners

What I Learned

Through this project, I learned how to design multi-role web interfaces, organize pages for different users, and improve my HTML and CSS skills while building a structured web application.
